# OSPF Routing with Wireshark Analysis

## Objective
- Configure OSPF on multiple routers
- Configure DHCP for LAN networks
- Verify end-to-end connectivity
- Analyze OSPF, DHCP, and ICMP using Wireshark

## Tools Used
- GNS3
- Cisco IOS
- Wireshark

## Lab Highlights
- Dynamic routing using OSPF
- DHCP across routed network
- Packet-level analysis
